<h1>volken</h1>

<div>
  <h3>목표</h3>
  perl로 만들어진 utility library.<br />
  추가 설치 권한이 없는 환경에서도 사용할 방법을 제공하는 것이 목표.
</div>

<div>
  <h3>기능</h3>
  <ul>
    <li>HTTP client - GET/POST</li>
    <li>HTTPS client - GET/POST (SSL 모듈 설치 필요)</li>
    <li>JSON 파서</li>
  </ul>
</div>
